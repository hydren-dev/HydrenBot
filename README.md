# HydrenBot
## This Bot Gives the Information of Your HydrenDashboard
## Main command /me 
**How to Install **
```bash
git clone https://github.com/hydren-dev/HydrenBot.git
```
```bash
cd HydrenBot
```
```npm
npm i axios dotenv
```
** Then Setup Everything which is in .env**
```bash
nano .env
```
# Then Enjoy Your bot
